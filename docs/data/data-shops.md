**목적**

상점(몰) 정보와 연동용 설정 예시를 제공합니다.

**예시 필드**

- `id`, `name`, `platform`, `isActive`, `settings`(object)

**예시**

```json
{
  "id":"shop-1",
  "name":"FULGO Official",
  "platform":"cafe24",
  "isActive": true,
  "settings": {"syncInventory": true}
}
```
