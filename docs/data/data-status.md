**목적**

여러 엔티티(주문/상품/재고 등)에 공통으로 사용하는 상태 코드와 의미를 정리합니다.

**예시**

- `ORDER_PLACED`: 주문 접수
- `ORDER_SHIPPED`: 배송 처리
- `PRODUCT_ACTIVE`: 판매중
