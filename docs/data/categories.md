스키마

- `id`: string — 예: `cat-1`
- `code`: string — 생성 규칙에 따른 코드
- `name`: string — 카테고리 이름

예시

```json
[
  { "id": "cat-1", "code": "CAT-20250913-7392-1", "name": "카테고리1" },
  { "id": "cat-2", "code": "CAT-20250913-2391-2", "name": "카테고리2" }
]
```
