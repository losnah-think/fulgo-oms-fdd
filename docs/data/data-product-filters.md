**목적**

프론트엔드에서 사용하는 상품 필터 스키마 예시(속성 필터, 옵션)를 정리합니다.

**예시**

```json
{
  "filters": [
    { "id":"color","label":"색상","values":["red","blue","green"] },
    { "id":"size","label":"사이즈","values":["S","M","L"] }
  ]
}
```
