**목적**

창고/로케이션 목업 스키마 예시를 제공합니다.

**스키마(예)**

- `id`: string
- `name`: string
- `location`: string

**예시**

```json
{ "id":"wh-1", "name":"서울 물류센터", "location":"서울시 ..." }
```

**비고**

출고 우선순위, 재고 배분 정책 등 물류 규칙을 문서화하세요.
